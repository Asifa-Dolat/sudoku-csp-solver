Project Description

This project solves Sudoku puzzles using Constraint Satisfaction Problem (CSP) techniques.
It applies:

Backtracking Search
Forward Checking
AC-3 Constraint Propagation
MRV (Minimum Remaining Values heuristic)

The goal is to efficiently solve Sudoku boards of different difficulty levels.
